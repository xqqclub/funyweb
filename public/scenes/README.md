# Scene Assets

Put image assets into these folders:

- `mobile/backgrounds/`
- `mobile/backgrounds/road/`
- `desktop/backgrounds/`
- `desktop/backgrounds/road/`
- `shared/buildings/`
- `shared/characters/`
- `shared/props/`

Suggested filenames:

- `mobile/backgrounds/city-office-day.png`
- `mobile/backgrounds/city-office-night.png`
- `mobile/backgrounds/forest-home-day.png`
- `mobile/backgrounds/forest-home-night.png`
- `mobile/backgrounds/road/road-day.png`
- `mobile/backgrounds/road/road-night.png`
- `desktop/backgrounds/city-office-day.png`
- `desktop/backgrounds/city-office-night.png`
- `desktop/backgrounds/forest-home-day.png`
- `desktop/backgrounds/forest-home-night.png`
- `desktop/backgrounds/road/road-day.png`
- `desktop/backgrounds/road/road-night.png`
- `shared/buildings/office-working.webp`
- `shared/buildings/home-going-home.webp`
- `shared/buildings/home-cleaning.webp`
- `shared/characters/main-working.webp`
- `shared/characters/main-going-home.webp`
- `shared/characters/main-cleaning.webp`
- `shared/props/effect-office-glow.webp`
- `shared/props/effect-commute-arrow.webp`
- `shared/props/effect-cleaning-bubbles.webp`

Road scene note:

- `going_home` and `biking` can share the same road background set.
- Put mobile road backgrounds in `mobile/backgrounds/road/`.
- Put desktop road backgrounds in `desktop/backgrounds/road/`.
- Use `road-day.png` for daytime and dusk.
- Use `road-night.png` for nighttime.

Use `config/scene-assets.json` as the starter layout mapping for mobile and desktop.
